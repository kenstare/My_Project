Project: My_Project

Overview
-
This workspace contains three small data projects focused on exploration, analysis, and simple model serving:

- `Eda_global_annmal_population/` — Exploratory Data Analysis and forecasting for world population.
- `Indian_Credit_card/` — Credit card transaction EDA and dashboard for India transactions.
- `Predicting_food_price_nigearia/` — Food price prediction models for Nigeria and a small API.

Quick setup
-
1. Create a virtual environment and install dependencies:

   Windows (PowerShell):

   ```powershell
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   pip install -r requirements.txt
   ```

2. Open the folder you want to work on and follow its `README.md` and `DOCUMENTATION.md`.

How to use this repo
-
- For exploratory work, open the Jupyter notebooks in each folder (files named `*.ipynb`).
- To run scripts, navigate to the target folder and run the listed `python` commands in that folder's README.

If you need help running a specific folder, open its `DOCUMENTATION.md` for detailed descriptions of files and data flow.
